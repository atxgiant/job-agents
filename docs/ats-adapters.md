# ATS Adapters

Initial adapter targets:

- Greenhouse
- Lever
- Ashby
- Workday
- Generic official careers page parser

## Contract

Adapters should expose discovery and job-detail fetching capabilities while preserving stable source identifiers, canonical URLs, and source metadata.

## Respectful behavior

- Use official sources only
- Respect rate limits
- Fail safely on unsupported or blocked sources
- Do not bypass authentication, CAPTCHAs, or anti-bot controls

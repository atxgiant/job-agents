# job-agents
tooling to search for jobs and develop tailored resumes that help me apply to the jobs I'm most competitive for

## Private skillset workflow

This repo includes a public `SKILLSET.md` template and expects your real private profile to live in `skillset.md`.

The local `skillset.md` file is gitignored so you can store private role history, quantified achievements, targeting logic, and other personal material without committing it.
